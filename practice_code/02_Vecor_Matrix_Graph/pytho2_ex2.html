벡터, 행렬 연산, 그래프 그리기¶

텐서란?¶

  • 다차원 배열을 일반화한 것
  • Scalar, vector, matrix 등이 포함됨

랭크란?¶

  • 텐서의 차원
  • Rank 0 tensor : Scalar
  • Rank 1 tensor : Vector
  • Rank 2 tensor : Matrix
  • Rank n tensor
